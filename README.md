# Roblox-Chat-Detection-Script
If you were wondering - no, this isn't a script meant for hacking. This is a script meant for Roblox game development. This script allows you to detect when a player chats a specific chat, and what to do when so.

Yes, you may use and repost this script. Loads of other people use it, but if you want to repost this script and you got it from here, please add my credit somewhere (in a readme file or in the script as a comment, in the description is great too).
